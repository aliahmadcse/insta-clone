# Instagram Clone

![Hero Image](/public/svg/hero.png)

## Features
- Create a Post
- Visit profiles
- Edit profile
- Timeline feature
- Profile section
- Follow/Unfollow a user

## Project setup
`composer install`

### Dependencies from composer.json
`npm install`

### Setting up .env File
`Set up .env file by creating one. A template is provided in .env.example`

### Make migration
`php artisan migrate`

### Serve
`php artisan serve`
